Como os testes foram feitos?

Os testes foram realizados de diversas formas no sistema. Uma delas foi a diferenciação de Cliente e Administrador, validando se o cliente realmente não consegue acessar o gerenciamento de produtos, categorias e cupons, através de testes de URL. Outro deles foi a validação de se era possível o cadastro de mais de um usuário com o mesmo nome, erro que atualmente está arrumado. Também houve a verificação do css, da obrigatoriedade de informações no formulário, e das relações das categorias com os produtos, como deletar um categoria que tem produtos pertencentes e o cadastro de produtos sem categoria. Procurando sempre por algo que poderia quebrar o sistema.

ERROS:

- CSS quebrado
- Cadastro de produtos faltando informações
- Deletar categoria que já tem um produto pertencente
- Cadastro de produtos sem categoria
